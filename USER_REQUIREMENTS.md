## Авторизация

- **Google OAuth ВЫНЕСЕН ЗА СКОУП MVP** — требует HTTPS, которого нет без домена
- MVP: авторизация-заглушка — один хардкоженный dev-пользователь (ID=1)
- Вся логика multi-tenant сохраняется в БД (UserId FK везде), но middleware просто возвращает UserId=1
- Это позволит добавить OAuth позже без изменения схемы БД
